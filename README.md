# nes

#### 介绍
c语言实现nes模拟器，要求c11

#### 软件架构
示例基于SDL进行图像声音输出，没有特殊依赖，可自行移植至任意硬件


#### 安装教程

​	windows下可选安装xmake / Visual Studio 进行编译

​	linux下需安装xmake进行编译

#### 使用说明

​	Visual Studio 不进行赘述

​	xmake方式需先安装[xmake](https://github.com/xmake-io/xmake) 随后直接执行 xmake 即可
